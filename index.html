Intro

Use case: Multiple test devices, multiple persons who want to run tests from their own machine

Redirector allows you to set your test url on all your devices at once for testing a certain site.

When set up you can use the following workflow:

  • Open up redirector frontend
  • Type in Url
  • Open up redirector frontend on testdevices
  • Get redirected to your page automatically

Using redirectors api you can even set the url automagically for your project.
For example you can define a task in gulp that starts browsersync sends its url to redirector.

More/better examples to come...
